The Atbash Cipher

The Atbash Cipher is purported to have been developed to encode the Hebrew Alphabet.

It is another mono-alphabetic cipher which takes the first letter in the alphabet and changes it to the last, the second for the second last, and so on. If we use the rule on the English alphabet, a becomes z, b becomes y, and so on.
